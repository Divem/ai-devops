## Context

需求评审看板是 PM-AI 平台的核心页面，用户在此进行需求管理和评审工作。当前页面右上角缺少设置入口，用户需要通过其他途径访问账号信息和项目配置，使用不便。

## Goals / Non-Goals

**Goals:**
- 在看板页面右上角添加设置图标按钮
- 实现下拉菜单，包含账号信息、项目配置、退出登录三个入口
- 保持与现有 UI 风格一致

**Non-Goals:**
- 不实现完整的用户设置页面
- 不实现项目配置的详细功能（仅提供跳转入口）

## Decisions

1. **下拉菜单实现方案**: 使用现有的 React 组件实现下拉菜单，参考项目中已有的下拉组件模式
2. **用户信息获取**: 从现有用户状态/上下文获取当前登录用户信息
3. **退出登录**: 调用现有的 logout 方法清除登录状态并重定向

## Risks / Trade-offs

- [风险] 新增的 UI 元素可能与现有页面布局冲突 →  mitigation: 使用绝对定位确保下拉菜单正确显示
- [风险] 用户未登录时不应显示设置入口 →  mitigation: 根据登录状态条件渲染
