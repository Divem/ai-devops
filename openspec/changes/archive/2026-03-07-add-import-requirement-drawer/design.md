## Context

看板页面右上角的"新建需求"按钮当前只有单一功能。用户需要能够从外部系统（如 Meego）导入已有需求到看板中。

## Goals / Non-Goals

**Goals:**
- 在"新建需求"按钮添加下拉箭头和下拉菜单
- 下拉菜单包含"新建需求"和"导入需求"两个选项
- 点击"导入需求"弹出右侧抽屉
- 抽屉支持 URL 导入和从列表选择导入

**Non-Goals:**
- 不实现真实的后端 API 调用（URL 导入为模拟）
- 不实现完整的空间/迭代/需求数据获取（使用模拟数据）

## Decisions

1. **下拉菜单实现**: 使用现有的 React 状态控制下拉菜单的显示/隐藏
2. **抽屉组件**: 参考项目中已有的 Drawer 组件实现风格
3. **选择器级联**: 空间选择 → 迭代选择 → 需求列表，三级级联选择

## Risks / Trade-offs

- [风险] 新增 UI 元素可能与现有页面布局冲突 →  mitigation: 使用相对定位确保下拉菜单正确显示
- [风险] 抽屉与详情面板冲突 →  mitigation: 导入抽屉与详情抽屉使用不同的触发逻辑
