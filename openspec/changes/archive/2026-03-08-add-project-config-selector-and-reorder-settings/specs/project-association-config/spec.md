## ADDED Requirements

### Requirement: 项目关联信息配置
系统 SHALL 在项目配置页面提供"项目配置"区块，支持用户选择要关联的空间、子系统和应用，并将选择结果保存到项目配置中。

#### Scenario: 选择空间后展示对应子系统
- **WHEN** 用户在"项目配置"区块选择一个空间
- **THEN** 系统仅展示该空间下可选的子系统列表

#### Scenario: 选择子系统后展示对应应用
- **WHEN** 用户已选择空间并继续选择一个子系统
- **THEN** 系统仅展示该子系统下可选的应用列表

#### Scenario: 保存项目关联配置
- **WHEN** 用户完成空间、子系统、应用选择并点击保存
- **THEN** 系统将三项选择持久化到 `projectConfig`，并显示保存成功提示

#### Scenario: 刷新后恢复项目关联配置
- **WHEN** 用户已保存项目关联配置并刷新页面
- **THEN** 系统从本地配置恢复空间、子系统、应用的已选值
